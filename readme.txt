这个应用程序的功能是将各种手机应用软件分类展示出来，如游戏软件,收费软件,免费软件等，可以方便、快速的找到想要的应用软件。参照91助手的界面布局、分类，软件的在主页面的信息展示功能，已经完成，为获得软件更详细的介绍，点击之后，显示详细介绍的界面也已经完成，还有显示其他信息的界面功能也已做好。有些功能还需以后继续完善。
下面将我在做此应用程序过程中的工作内容和用到的专业技术作简单介绍：1、将整体的显示界面做好之后，使用第三方库AFNetworking发送数据请求，获取服务器端的JSON数据并解析；
2、为了界面的美观，多次使用ScrollView及其代理自动动态轮播图片，或者手动滑动应用图片；
3、自定义cell，将要显示的按钮，标签位置设置好，将获取的数据，显示在对应的位置；
4、按钮上的图片和UIImageView上的图片使用SDWebImage第三方开源库异步请求下载图片; 5、不同类中，功能一样，需要多次重复使用的代码进行合并，做一些简单的优化，便于维护，修改；
